全局变量在数据段内，代码在代码段，还有栈和堆

栈：

每个程序、每个函数都有自己的栈，进入程序、函数时栈空间已经被分配好，栈大小由内部定义的变量的数量和大小及其他（返回地址等）确定。
在循环中创建的对象（非new方法），进入下一次循环就被释放，只有通过拷贝构造的方法复制出去（进入某个数据集或存到某个外部变量）。

P p; //变量在外
while(){
    p = P();
    vector1.push_back(p); 可以，通过拷贝构造复制出去
    vector1.push_back(&p); 不可以，每次地址都一样，所有指针都指向最后一次创建的那个对象
}

while(){
    P p; //变量在内
    vector1.push_back(p); 可以，通过拷贝构造复制出去
    vector1.push_back(&p); 不可以，所有指针同指向一片未知内存，p出循环调用析构被释放（可用P*访问得到值，内存没变，标记变了）
}
